# 2. 学生端：课程修读与成绩查询

## 一、基本文件结构

```
student-portal/
│
├── login.html                     # 登录页面
├── index.html                     # 学生主页（登录后）
├── course-browse.html             # 课程浏览与选修
├── my-courses.html                # 我的课程列表
├── course-detail.html             # 课程学习页面
├── grades.html                    # 成绩查看页面
│
├── css/
│   └── style.css                  # 所有样式
│
└── js/
    ├── api.js                     # API调用封装
    ├── auth.js                    # 登录状态管理
    └── main.js                    # 通用工具函数
```

## 二、页面功能说明

### 1. index.html - 学生主页
- 顶部导航栏（我的课程、成绩查询、退出登录）
- 欢迎信息（学生姓名）
- 三个大按钮：
  - "浏览课程" → 跳转到 course-browse.html
  - "我的课程" → 跳转到 my-courses.html  
  - "查看成绩" → 跳转到 grades.html

### 2. course-browse.html - 课程浏览
- 搜索框（按课程名搜索）
- 课程列表（卡片式展示）
- 每张卡片显示：
  - 课程名称
  - 授课教师
  - 课程学分
  - "选修"按钮
- 点击"选修"弹出确认框，确认后调用选课API

### 3. my-courses.html - 我的课程
- 学期选择下拉框
- 课程列表（表格形式）
- 每行显示：
  - 课程名称
  - 授课教师  
  - 学期
  - "详情"按钮 → 跳转到 course-detail.html

### 4. course-detail.html - 课程学习
- 课程基本信息（名称、教师）
- 学习任务列表（表格）
- 每行显示：
  - 任务名称（如：第一章课件）
  - 任务类型（文档/视频）
  - 状态（待完成/已完成）
  - 操作按钮（查看/标记完成）

### 5. grades.html - 成绩查看
- 学期选择下拉框
- 成绩总览（表格）：
  - 课程名称
  - 学分
  - 成绩
  - GPA
- 点击某课程行 → 显示该课程成绩详情
  - 各成绩项及分数
  - 总分计算

## 三、主要功能对应API

1. **获取课程列表** - `GET /api/v1/courses`
2. **选修课程** - `POST /api/v1/enrollments`  
3. **查看我的课程** - `GET /api/v1/me/enrollments`
4. **查看学习任务** - `GET /api/v1/me/enrollments/{id}/tasks`
5. **查看成绩总览** - `GET /api/v1/me/grades/summary`
6. **查看课程成绩** - `GET /api/v1/me/enrollments/{id}/grades`
